# 🎮 프리미엄 게임 품질 바 (Premium Game Quality Bar)

> 윈큐브 게임 데모 = B2B 영업 시연 도구. "돈 주고 파는 느낌"이 기준이다.
> 모든 게임 작업(신규·고도화) 전·중·후에 이 문서를 강제 적용한다.
> 작성 계기: 2026-06-26 — 구버전을 붙잡고 작업해 DX판과 품질 격차가 난 사건.

---

## 0단계 — 작업 전 탐색 (구버전 작업 방지) ⚠️ 가장 자주 빠뜨림

게임 파일을 열기 **전에** 반드시:
1. 같은 게임의 다른 버전을 Glob으로 전수 탐색한다.
   - 예: `**/*.html`에서 동일 게임명, `ZCodeProject/`, 백업 폴더 등
2. 후보들의 **수정일(LastWriteTime)·줄수·크기**를 비교한다.
3. **가장 진보한 버전을 베이스로 채택**한다. 오래된 걸 고도화하지 않는다.
4. 애매하면 사용자에게 "A판/B판 중 어느 걸 베이스로?"라고 1번 묻는다.

> 교훈: "내가 받은 파일 = 최신"이라고 가정 금지. 항상 더 나은 버전을 먼저 찾는다.

---

## 품질 바 — 7개 축 (돈 주고 파는 느낌)

작동만 하는 건 50점이다. 아래를 갖춰야 "상품" 수준이다.

### 1. 게임 필 / Juice (체감 타격감)
- **스쿼시 & 스트레치** — 점프·착지·매치 순간 눌림/늘어남
- **스크린 셰이크** — 임팩트 크기에 비례 (작은 건 약하게, 큰 건 강하게)
- **히트스톱** — 결정적 순간(사망·대박) 몇 프레임 정지 → 묵직함
- **펀치 줌** — 피버 진입·대박 시 살짝 줌인
- **파티클** — 먼지/폭발/스파크, 이벤트마다 다른 종류
- **화면 플래시 / 비네트** — 피격(빨강)·고조(금색)

### 2. 보상 루프
- **콤보** + 데케이(시간 지나면 리셋=긴장감) + 배수
- **피버/절정 모드** — 게이지 차면 발동, 시각이 완전히 달라짐(레인보우/무적/BGM가속)
- **마이크로 목표** — 항상 "다음 작은 성취"가 보일 것

### 3. 오디오 (Web Audio 합성, 에셋 0)
- 행동마다 **구분되는 SFX** (점프/획득/실패/대박)
- **동적 BGM** — 속도·피버에 따라 레이어/템포 변화
- **음소거 토글** 필수

### 4. 진행감 / 난이도
- 거리·점수에 따른 **페이즈 전환** + 시각 변화(팔레트/테마/시간대)
- 난이도 곡선: 초반 워밍업 → 후반 가속

### 5. 영속성 (Persistence) ⚠️ DX판이 빠뜨렸던 치명적 결함
- **최고기록/진행을 localStorage에 저장** — 새로고침해도 살아있을 것
- 신기록 시 특별 연출

### 6. 폴리시
- 메뉴·결과 화면에 캐릭터/개성 (밋밋한 텍스트 금지)
- "신기록!" 축하, 일시정지/재시작 매끄럽게
- 모바일 터치 + 키보드 동시 지원

### 7. 검증 (증거 기반, 빼면 완료 아님)
- **Codex read-only 더블체크** (VERIFY_OK)
- **Chrome DevTools 실플레이** — 콘솔 에러 **0** 확인
- 핵심 기능을 evaluate_script로 실제 구동해 결과값 확인

## 적용 체크리스트 (작업 종료 전 자가 점검)

- [ ] 0단계 탐색으로 최신 베이스를 골랐는가
- [ ] 7개 축을 모두 점검했는가 (없는 축은 의도적으로 생략했는가)
- [ ] localStorage 저장이 새로고침 후에도 동작하는가
- [ ] 콘솔 에러 0인가 (실플레이로 확인)
- [ ] Codex VERIFY_OK 받았는가
- [ ] 메뉴/결과 화면이 "상품" 같은가, 아니면 개발자 플레이스홀더 같은가
